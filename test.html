Example usage:
--------------

var my_datepicker = jrDatePicker({
    dp_id_name: 'disp_datepicker',    // datepicker is displayed here
    id_name: 'start_date',            // selected date is inserted here
    locale: 'fr',                     // setting locale to french
    onDateSelected: function() {alert("ondateselected callback");},
    onClose: function() {alert("onclose callback");}
});


Check-Out Date: <input type="text" id="start_date" value="" />
                <img class="jrdp_calendar_icon" src="icon_calendar.gif" onclick="my_datepicker.show();" />
                <span class="jrdp_calendar_pos" id="disp_datepicker">


Options:

    dp_id_name:      location where the datepicker is to be displayed - required
    id_name:         location where the selected date is copied - optional
    locale:          en, fr, es, de - default is 'en'
    display_count:   number of months to display in datepicker - default is 1
    close_onselect:  true | false - default is true
    max_date:        maximum date user can scroll forward to and select - default is 1Y
                     acceptable values: 3M, 6M, 9M, 1Y, 2Y, * (infinity)
    min_date:        minimum date user can scroll backward to and select - default is 0 (current date)
                     acceptable values: 0, 3M, 6M, 9M, 1Y, 2Y, * (infinity)
    onDateSelected:  callback function
    onClose:         callback function
