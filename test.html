jrDatePicker is open source and available at github here.

Example usage:
--------------

var my_datepicker = jrDatePicker({
    dp_id_name: 'disp_datepicker',    // datepicker is displayed here
    id_name: 'start_date',            // selected date is inserted here
    locale: 'fr',                     // setting locale to french
    max_date: '3M',                   // maximum selectable date
    min_date: '3M',                   // minimum selectable date
    onDateSelected: function() {alert("ondateselected callback");},
    onClose: function() {alert("onclose callback");}
});

Files
jrDatePicker.js Javascript code for the datepicker.
jrDatePicker.css Styling for the datepicker. You can provide you own styling however the class names are required.
 
Options
dp_id_name location where the datepicker is to be displayed
required
id_name location where the selected date is copied
optional
locale en, fr, es, de
default is 'en'
display_count number of months to display in datepicker
default is 1
close_onselect true | false
default is true
max_date maximum date user can scroll forward to and select
default is 1Y
acceptable values: 3M, 6M, 9M, 1Y, 2Y, * (infinity)
min_date minimum date user can scroll backward to and select
default is 0 (current date)
acceptable values: 0, 3M, 6M, 9M, 1Y, 2Y, * (infinity)
onDateSelected callback function
onClose callback function
